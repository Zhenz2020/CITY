# 决策功能调试指南

## 如果还是看不到决策输出

### 步骤1: 重启后端

在运行后端的终端中：
```
按 Ctrl+C 停止
然后运行: python backend/app.py
```

### 步骤2: 测试决策 API

打开浏览器访问测试页面：
```
file:///D:/项目/CITY/test_websocket.html
```

然后点击"测试获取决策"按钮，看是否能收到数据。

### 步骤3: 检查控制台

在前端页面按 F12 打开开发者工具：
1. 查看 Console 是否有错误
2. 点击车辆后，看 Network 标签页是否有请求发送到 `/api/agent/xxx/decision`
3. 查看 WebSocket 连接状态

### 步骤4: 验证 LLM 配置

确保 `config/siliconflow_config.json` 中有有效的 API Key：
```json
{
  "api_key": "sk-你的真实API密钥",
  "model": "Qwen/Qwen3-14B"
}
```

如果没有配置 LLM，系统会返回规则决策（如 "ACCELERATE" / "DECELERATE"）。

## 预期行为

1. 点击画布上的车辆 → 实体详情面板显示车辆信息
2. 点击"获取决策"按钮 → 右侧面板显示 AI 决策输出
3. 决策输出包含：
   - 大标题显示决策动作（如"加速"）
   - 决策理由
   - 环境感知（JSON 格式）
   - 完整决策数据

如果没有配置 LLM，会显示规则决策（如 action: "ACCELERATE"）。
