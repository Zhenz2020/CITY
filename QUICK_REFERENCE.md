# CITY 系统 - 快速参考卡片

## 🚀 启动命令

```bash
# 一键启动（推荐）
python start_all.py

# 或分别启动
python backend/app.py              # 终端1
cd frontend/city-frontend && npm start   # 终端2
```

**访问地址**: http://localhost:3000

---

## 🎮 常用操作

| 操作 | 步骤 |
|------|------|
| **启动仿真** | 点击左侧【开始】按钮 |
| **暂停仿真** | 点击左侧【暂停】按钮 |
| **生成车辆** | 选择类型 → 点击【生成车辆】|
| **查看车辆** | 点击画布上的车辆 |
| **获取AI决策** | 选中车辆 → 点击【获取决策】|
| **重置系统** | 点击【重置】按钮 |

## 🔧 配置 LLM

**文件**: `config/siliconflow_config.json`

```json
{
  "api_key": "sk-你的API密钥",
  "model": "Qwen/Qwen3-14B"
}
```

获取 API Key: https://siliconflow.cn

## ❗ 常见问题

### 页面显示"未连接"
```bash
# 确保后端已启动
python backend/app.py
```

### LLM决策返回"服务不可用"
1. 检查 API Key 是否正确
2. 检查网络连接

### 画布空白
1. 按 F12 查看错误
2. 刷新页面
3. 等待几秒让数据加载
