## 快速启动

### 1. 安装依赖

```bash
# 1. 安装后端依赖
pip install flask flask-socketio flask-cors

# 2. 安装前端依赖
cd frontend/city-frontend
npm install
cd ../..
```

### 2. 启动系统

**一键启动（推荐）:**
```bash
python start_all.py
```

或分别启动:
```bash
# 终端1: 启动后端
python backend/app.py

# 终端2: 启动前端
cd frontend/city-frontend
npm start
```

### 3. 访问界面

打开浏览器: http://localhost:3000

## 核心功能

### 1. 可视化画布 (PixiJS)
- ✅ 实时显示道路网络
- ✅ 车辆移动动画（不同颜色代表不同车型）
- ✅ 行人显示（橙色圆点）
- ✅ 交通信号灯状态（红黄绿）
- ✅ 点击选中实体

### 2. 控制面板
- ✅ 开始/暂停/重置仿真
- ✅ 生成车辆（轿车、公交、卡车、应急车、摩托车）
- ✅ 实时统计（时间、车辆数、完成率）

### 3. 实体详情面板
- ✅ 显示选中实体的详细信息
- ✅ 实时位置、速度、状态
- ✅ **LLM决策输出显示**（感知 + 决策 + 理由）
- ✅ 刷新决策按钮

### 4. 决策日志
- ✅ 滚动显示所有实体的决策历史
- ✅ 时间戳、感知信息、决策结果

## 使用流程

1. **启动系统** - 运行 `python start_all.py`
2. **开始仿真** - 点击"开始"按钮
3. **生成车辆** - 选择车型并点击"生成车辆"
4. **查看详情** - 点击画布上的车辆
5. **获取决策** - 点击"获取决策"查看LLM输出
6. **查看日志** - 在决策日志面板查看历史
