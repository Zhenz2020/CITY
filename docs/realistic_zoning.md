# 现实城市规划系统 (Realistic Urban Planning)

## 概述

本系统实现了基于真实城市规划原则的功能区域规划，模拟现实中城市发展的逻辑和约束。

## 核心特性

### 1. 服务半径约束 (Service Radius)

不同公共设施有特定的服务半径要求：

| 设施类型 | 服务半径 | 说明 |
|---------|---------|------|
| 学校 | 500米 | 学生步行可达范围 |
| 医院 | 2000米 | 急救车辆响应范围 |
| 公园 | 800米 | 居民日常休闲范围 |
| 商业区 | 1000米 | 日常生活服务范围 |

系统会自动评估新设施的服务覆盖范围，确保选址能够服务足够的人口。

### 2. 区域兼容性 (Zone Compatibility)

不同区域类型之间有兼容性约束：

**住宅区兼容：**
- ✅ 商业区、公园、学校、办公区、购物中心、混合区
- ❌ 工业区（噪音和污染）

**工业区兼容：**
- ✅ 商业区、办公区、政府机构
- ❌ 住宅区、学校、医院（需要保持安全距离）

**医院兼容：**
- ✅ 住宅区、公园、商业区
- ❌ 工业区（环境影响）

### 3. 最小间距约束 (Minimum Spacing)

同类型设施之间需要保持最小距离：

| 设施类型 | 最小间距 | 原因 |
|---------|---------|------|
| 学校 | 300米 | 避免教育资源过度集中 |
| 医院 | 1000米 | 医疗资源配置均衡 |
| 公园 | 200米 | 合理分布绿地 |
| 商业区 | 150米 | 避免恶性竞争 |

### 4. LLM辅助决策

系统使用LLM（大语言模型）进行智能选址评估：

**评估维度：**
1. **兼容性评分** - 与周边区域类型的匹配程度
2. **服务覆盖评分** - 目标人口的覆盖比例
3. **道路可达性** - 距离道路网络的远近
4. **间距合理性** - 与同类型设施的距离

**LLM决策流程：**
1. 生成多个候选位置
2. 对每个候选进行多维度评估
3. 使用LLM评估选址合理性
4. 综合评分选择最优解
5. 如果LLM不批准且评分过低，放弃该选址

### 5. 人口密度模型

不同区域类型有不同的人口容量：

| 区域类型 | 人口密度 | 说明 |
|---------|---------|------|
| 住宅区 | 50人/1000m² | 高密度居住 |
| 商业区 | 20人/1000m² | 商业活动 |
| 办公区 | 40人/1000m² | 办公就业 |
| 学校 | 100人/1000m² | 学生+教职工 |
| 医院 | 30人/1000m² | 医护人员+患者 |
| 工业区 | 10人/1000m² | 工厂工人 |
| 公园 | 5人/1000m² | 休闲游客 |

## 规划策略

### 住宅区规划

**选址策略：**
- 分散布置，避免过度集中
- 优先靠近公园和学校
- 远离工业区和主要交通干道

**配套要求：**
- 500米范围内应有学校
- 400米范围内应有公园
- 300米范围内应有商业设施

### 商业区规划

**选址策略：**
- 靠近住宅区，服务消费群体
- 交通便利，可达性高
- 与其他商业区保持适当距离

**评估指标：**
- 附近住宅区的数量和人口
- 道路可达性评分
- 与现有商业区的距离

### 学校规划

**选址策略：**
- 靠近住宅区，方便学生上学
- 与其他学校保持300米以上距离
- 远离工业区，环境安全

**服务评估：**
- 500米半径内的住宅人口
- 是否有足够的适龄学生
- 交通便利性

### 医院规划

**选址策略：**
- 城市中心或交通便利位置
- 与其他医院保持1公里以上距离
- 服务半径覆盖足够人口

### 公园规划

**选址策略：**
- 靠近住宅区，方便居民使用
- 与其他公园保持适当分布
- 环境兼容性好

### 工业区规划

**选址策略：**
- 城市边缘位置
- 远离住宅区、学校、医院
- 交通便利，便于物流

**安全距离：**
- 与住宅区至少300米
- 与学校、医院至少500米

## 使用示例

```python
from city.agents.planning_agent import PopulationCityPlanner
from city.simulation.environment import SimulationEnvironment

# 创建城市规划智能体
planner = PopulationCityPlanner(
    environment=env,
    use_llm=True,               # 启用LLM辅助决策
    enable_zoning=True,         # 启用城市规划
    zoning_interval=20.0,       # 每20秒尝试规划
    max_zones=30                # 最大区域数
)

# 添加到仿真环境
env.add_agent(planner)
```

## 未来优化方向

1. **动态需求模型** - 根据人口增长动态调整规划需求
2. **交通影响评估** - 评估新区域对交通流量的影响
3. **环境容量模型** - 考虑环境承载力的规划限制
4. **多目标优化** - 平衡经济效益、社会效益、环境效益
5. **历史学习** - 从规划历史中学习和优化策略
